# 00 Start Here

This folder is an Obsidian-style project knowledge base for the [[Engineering Simulation Toolkit]].

The goal is to give a new ChatGPT Project chat, Codex session, or human reviewer enough context to work safely without rereading the full repository history.

## Project Goal

Build a clear educational/prototype/portfolio engineering simulation toolkit moving toward professional quality. The project demonstrates ODE/PDE modeling, control systems, numerical methods, UAV/quadcopter examples, state estimation, FEM basics, scientific visualization, tests, examples, docs, and Streamlit demos.

> [!warning]
> Do not frame this as production-grade, certified, flight-ready, or industrial simulation software.
