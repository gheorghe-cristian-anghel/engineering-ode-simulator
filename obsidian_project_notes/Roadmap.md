# Roadmap

The roadmap is professionalization-focused: stabilize, validate, polish, then expand.

## Short-Term

- Keep tests green.
- Fix confirmed bugs only.
- Regression-test Streamlit plot display after helper changes.
- Improve documentation and handoff context.
- Polish graph readability, labels, captions, and legends.

## Medium-Term

- Continue scientific validation tests.
- Add more analytical comparisons where available.
- Improve Streamlit PDE usability without redesigning the app.
- Curate screenshots and selected animations.
- Add report/export polish for portfolio demos.

## Future Professionalization

- Packaging and release polish.
- Optional documentation site.
- Reproducible gallery asset generation.
- More formal benchmark/validation references.
- 2D truss FEM solver.
- Beam bending FEM.

## Future Advanced Features

Only after stability and validation:

- richer FEM examples
- more animation exports
- optional reports
- broader state-estimation demos
- additional controls examples
